## X API

#### Run app
```shell
cp .env.example .env
npm i
npm run start
```

#### Env variables
| Env variable name | Required | Default | Description                |                                                                                                                                                                                                                                                                                                                                                                                                                                                
|-------------------|----------|---------|----------------------------|
| DATABASE_URL      | true     | -       | Database connection string |
